## Environments

- `Node`: `>= 8.10.0`
- `npm`: `>= 6`
- `yarn`: `>= 1`

## Usage

### Install(optional)

```bash
npm install create-cha -g
```

### Use

- Use with `npm init`(**recommend**)

```bash
npm init cha <project-name>
# or
npm create cha <project-name>
```

- Use with `npx`

```bash
npx create-cha <project-name>
```

- Use directly

```bash
create-cha <project-name>
```

more use details see `create-cha -h`
